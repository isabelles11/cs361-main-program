## Overview
This repository contains the Main Program for the CS361 course project. The project is a medication reminder application designed to help users create medication schedules, view upcoming doses, and log whether doses are taken or skipped.

At this stage, this repository primarily contains planning and setup artifacts required for Milestone #1. The functional implementation will be developed incrementally in later milestones.

## Project Goals
- Provide a simple and inclusive way for users to manage medication reminders
- Support usability, accessibility, and reliability as core quality attributes
- Serve as the integration point for future microservices

## Current Status
- GitHub repository initialized
- Test commits completed to verify environment setup
- Project planning and user stories in progress

## Planned Features
- Add and manage medications
- View daily medication schedules
- Log doses as taken or skipped
- Integrate with microservices for reminders and analytics (future work)

## Technologies
To be determined. The technology stack will be selected based on course requirements and project scope.

## Notes
This application is a reminder and logging tool only and does not provide medical advice.
